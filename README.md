# The SNK Desktop (Kernel)

Репозиторий, который пока что является реализацией The SNK Kernel под названием The SNK Desktop. В Будущем будет являться полноценным отедельным репозиторием ядра The SNK Kernel, представляющий модульную абстракцию над Оборудованием(Hardware Abstraction Layer)/Системой управления программами(Program Managment)/Другими модулями.
